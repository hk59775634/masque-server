# AFBuyers MASQUE VPN 正式发布（Phase 1）

今天我们正式完成 AFBuyers MASQUE VPN 的 Phase 1（M1-M4）里程碑交付。  
这意味着：从账号接入、策略管控到监控告警与上线验收，核心闭环已具备生产可用能力。

## 本次发布带来了什么？

- **更易用的接入体验**：支持用户注册登录、设备激活、Linux 客户端一键连接与断开恢复。
- **更精细的访问控制**：支持用户/设备维度策略下发，服务端 ACL 按网段/协议/端口精确放行。
- **更可靠的运维能力**：内置 Prometheus + Grafana + Alertmanager 可观测体系，支持告警联通与自动化巡检脚本。
- **更安全的管理操作**：高危操作升级为一次性确认码（短时有效、单次使用），支持强制下线与批量会话治理。
- **更可审计的合规能力**：审计中心支持筛选、导出、变更 diff，并新增防篡改哈希链校验机制。

## 对客户价值

- **上线更快**：提供部署/回滚脚本与标准化运行手册。  
- **风险更低**：关键高危操作有二次确认与全量审计留痕。  
- **定位更准**：指标、日志、告警三位一体，问题发现与排障更高效。

## 适用场景

- 企业/团队远程访问内网资源  
- 开发与运维环境的细粒度网络访问管控  
- 需要审计留痕与操作可追溯的安全场景

- 更完整证书体系与高级认证能力  
- 多节点与高可用架构能力演进
